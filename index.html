PING PONG

Welcome! This web page will create a web application that takes a number from a user and returns a range of numbers from 1 to the
chosen number with the following exceptions:

  • Numbers divisible by 3 are replaced with "PING"
  • Numbers divisible by 5 are replaced with "PONG"
  • Numbers divisible by 15 are replaced with "PING-PONG"
